# Markdown简要操作手册

> "简洁是终极的复杂。"

## 什么是Markdown

Markdown是一种轻量级的**标记语言**，由John Gruber于2004年创建。它允许人们使用易读易写的纯文本格式编写文档，然后将其转换为有效的HTML文档。

## 为什么学习Markdown

- **简洁易学**：语法规则简单，几分钟即可上手；
- **广泛支持**：GitHub、微信公众号和Agent工具等均支持；
- **纯文本格式**：不依赖特定软件，任何文本编辑器都能编写；
- **专注内容**：无需频繁使用鼠标，让写作更流畅。
