# 第二章 扩展语法

> "经常不断地学习，你就什么都知道。你知道得越多，你就越有力量。"

本章介绍Markdown的扩展语法，这些功能在GitHub、GitLab等平台上得到广泛支持。

---

## 2.1 任务列表

使用 `- [ ]` 表示未完成任务，`- [x]` 表示已完成任务。

**语法：**
```markdown
- [x] 完成项目文档
- [x] 提交代码审查
- [ ] 修复Bug #123
- [ ] 部署到生产环境
```

**效果：**
> - [x] 完成项目文档
> - [x] 提交代码审查
> - [ ] 修复Bug #123
> - [ ] 部署到生产环境

**适用场景：** 项目待办清单、学习计划、会议记录等。

---

## 2.2 脚注

使用 `[^标识]` 添加脚注引用，在文档末尾定义脚注内容。

**语法：**
```markdown
这里需要一个脚注[^1]，这里也需要[^note]。

[^1]: 这是第一个脚注的内容。
[^note]: 这是一个命名脚注。
```

**效果：**
> 这里需要一个脚注[^1]，这里也需要[^note]。
>
> [^1]: 这是第一个脚注的内容。
> [^note]: 这是一个命名脚注。

---

## 2.3 定义列表

使用 `:` 符号创建定义列表。

**语法：**
```markdown
术语1
: 术语1的定义

术语2
: 术语2的定义
: 术语2的另一个定义
```

**效果：**
> 术语1
> : 术语1的定义
>
> 术语2
> : 术语2的定义
> : 术语2的另一个定义

---

## 2.4 自动链接

用尖括号 `< >` 包裹URL或邮箱地址，会自动转换为可点击的链接。

**语法：**
```markdown
<https://github.com>
<user@example.com>
```

**效果：**
> <https://github.com>
> <user@example.com>

---

## 2.5 转义字符

使用反斜杠 `\` 转义特殊字符，使其显示为普通字符。

**可转义的字符：**
```
\   反斜杠
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井号
+   加号
-   减号
.   句号
!   感叹号
|   竖线
```

**语法：**
```markdown
\*这不是斜体\*
\[这不是链接\]
```

**效果：**
> \*这不是斜体\*
> \[这不是链接\]

## 2.7 数学公式

使用 `$` 包裹行内公式，使用 `$$` 包裹独立公式块。

**语法：**
```markdown
行内公式：$E = mc^2$

独立公式块：
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
```

**效果：**
> 行内公式：$E = mc^2$
>
> 独立公式块：
> $$
> \sum_{i=1}^{n} i = \frac{n(n+1)}{2}
> $$

## 2.8 折叠内容

使用 `<details>` 和 `<summary>` HTML标签创建可折叠内容。

**语法：**
```markdown
<details>
<summary>点击展开详情</summary>

这里是隐藏的内容。

- 可以包含列表
- 代码块
- 等各种内容

</details>
```

**效果：**
<details>
<summary>点击展开详情</summary>

这里是隐藏的内容。

- 可以包含列表
- 代码块
- 等各种内容

</details>

## 2.9 键盘按键

使用 `<kbd>` 标签表示键盘按键。

**语法：**
```markdown
按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 复制
```

**效果：**
> 按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 复制

---

## 2.10 警告提示框

GitHub支持的特殊引用语法，用于创建不同类型的提示框。

**语法：**
```markdown
> [!NOTE]
> 这是一个注释提示框。

> [!TIP]
> 这是一个提示框。

> [!WARNING]
> 这是一个警告框。

> [!CAUTION]
> 这是一个警告框。
```

**效果：**
> [!NOTE]
> 这是一个注释提示框。

> [!TIP]
> 这是一个提示框。

> [!WARNING]
> 这是一个警告框。

> [!CAUTION]
> 这是一个警告框。
