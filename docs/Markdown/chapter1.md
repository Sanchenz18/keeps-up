# 第一章 基础语法

> "工欲善其事，必先利其器。"

本章介绍Markdown最常用的语法，掌握这些内容即可满足日常写作需求。

---

## 1.1 标题

使用 `#` 号标记标题，一个 `#` 是一级标题，两个 `#` 是二级标题，以此类推，最多支持六级标题。

**语法：**
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**
> # 一级标题
> ## 二级标题
> ### 三级标题
> #### 四级标题
> ##### 五级标题
> ###### 六级标题

**注意：** `#` 后面需要加一个空格。

---

## 1.2 段落与换行

### 段落

段落之间使用**空行**分隔。

**语法：**
```markdown
这是第一段内容。

这是第二段内容。
```

### 换行

在行末添加**两个空格**或使用 `<br>` 标签实现换行。

**语法：**
```markdown
这是第一行。  
这是第二行。
```

---

## 1.3 强调

### 粗体

使用 `**` 或 `__` 包裹文本。

**语法：**
```markdown
**这是粗体**
__这也是粗体__
```

**效果：**
> **这是粗体**
> __这也是粗体__

### 斜体

使用 `*` 或 `_` 包裹文本。

**语法：**
```markdown
*这是斜体*
_这也是斜体_
```

**效果：**
> *这是斜体*
> _这也是斜体_

### 粗斜体

使用 `***` 包裹文本。

**语法：**
```markdown
***这是粗斜体***
```

**效果：**
> ***这是粗斜体***

### 删除线

使用 `~~` 包裹文本。

**语法：**
```markdown
~~这是删除线~~
```

**效果：**
> ~~这是删除线~~

---

## 1.4 列表

### 无序列表

使用 `-`、`*` 或 `+` 开头。

**语法：**
```markdown
- 第一项
- 第二项
- 第三项
```

**效果：**
> - 第一项
> - 第二项
> - 第三项

### 有序列表

使用数字加 `.` 开头。

**语法：**
```markdown
1. 第一项
2. 第二项
3. 第三项
```

**效果：**
> 1. 第一项
> 2. 第二项
> 3. 第三项

### 嵌套列表

在子项前添加两个空格或一个制表符。

**语法：**
```markdown
- 第一项
  - 子项 A
  - 子项 B
- 第二项
```

**效果：**
> - 第一项
>   - 子项 A
>   - 子项 B
> - 第二项

---

## 1.5 链接

### 行内链接

**语法：**
```markdown
[链接文字](URL)
[GitHub](https://github.com)
```

**效果：**
> [GitHub](https://github.com)

### 带标题的链接

**语法：**
```markdown
[链接文字](URL "标题")
[GitHub](https://github.com "点击访问GitHub")
```

**效果：**
> [GitHub](https://github.com "点击访问GitHub")

### 引用链接

**语法：**
```markdown
[链接文字][引用标识]

[引用标识]: URL
```

---

## 1.6 图片

**语法：**
```markdown
![替代文字](图片URL)
![替代文字](图片URL "图片标题")
```

**示例：**
```markdown
![Logo](logo.png "网站Logo")
```

**注意：** 替代文字在图片无法显示时会显示出来。

---

## 1.7 代码

### 行内代码

使用反引号 `` ` `` 包裹代码。

**语法：**
```markdown
使用 `print()` 函数输出内容。
```

**效果：**
> 使用 `print()` 函数输出内容。

### 代码块

使用三个反引号 ` ``` ` 包裹代码块，可指定语言实现语法高亮。

**语法：`
```markdown
```python
def hello():
    print("Hello, World!")
```
```

**效果：**
```python
def hello():
    print("Hello, World!")
```

---

## 1.8 引用

使用 `>` 符号标记引用内容。

**语法：**
```markdown
> 这是一段引用内容。
> 
> 这是引用的第二段。
```

**效果：**
> 这是一段引用内容。
> 
> 这是引用的第二段。

### 嵌套引用

**语法：**
```markdown
> 第一层引用
>> 第二层引用
>>> 第三层引用
```

**效果：**
> 第一层引用
>> 第二层引用
>>> 第三层引用

---

## 1.9 表格

使用 `|` 分隔列，使用 `-` 分隔表头。

**语法：**
```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
```

**效果：**

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |

### 对齐方式

**语法：**
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   | 内容     | 内容   |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   | 内容     | 内容   |
